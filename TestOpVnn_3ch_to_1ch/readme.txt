Демонстрационное приложение, показывающее пример использования библиотеки OpenVINO для
запуска готовой нейронной сети. Не обучает сеть, только запускает процедуру распознавания.

Получает на вход 3-канальное изображение, на выходе 1-канальное изображение.

Нейронная сеть обучалась выявлять на зашумлённом изображении эллипсы (вертикально
или горизонтально ориентированные).

Проект создан в Microsoft Visual Studio 2017.

Чтобы собрать проект вам необходимы установленные в системе библиотеки OpenCV и OpenVINO.

Загружает нейронную сеть из файлов:
    zf_unet_224.bin (веса)
	zf_unet_224.xml (топология)

И пропускает через неё изображение:
    input.png

Результат работы сети сохраняет в файл:
    output_image.png

В папке /data/ сохранено входное изображение и то, что должно получиться в результате
работы нейронной сети.

Вы можете создать свои примеры входных данных. Для этого создайте цветное изображение,
нарисуйте на нём эллипс (резко более светлого тона, чем фон), сделайте сильно более
тёмный фон, затем на получившемся изображении нарисуйте не очень большое количество
случайных точек случайного цвета. Пример можно посмотреть в папке /data/. Сохраните
ваше изображение в формате png в размере 224 на 224 и переименуйте в input.png, затем
поместите рядом с бинарником. Убедитесь, что файлы нейронной сети находятся также рядом.
Запустите TestOpVnn.exe, результат работы сети будет помещён в файл output_image.png.

Не забывайте о том, что данное приложение является демонстрационным, поэтому качество
распознавания эллипсов может быть очень плохим (сеть обучалась не очень долго). Для
промышленного применения данное приложение непригодно.
