Демонстрационное приложение, показывающее пример использования библиотеки OpenVINO для
запуска готовой нейронной сети. Не обучает сеть, только запускает процедуру распознавания.

Получает на вход 1-канальное изображение, на выходе 3-канальное изображение (задача сегментации
с 3-мя видами областей, нераспознаное помечается чёрным).

По умолчанию используется нейронная сеть для сегментации медицинских снимков (сеть и данные
недоступны по причине лицензии).

Проект создан в Microsoft Visual Studio 2017 (c++ 17 стандарта).

Чтобы собрать проект вам необходимы установленные в системе библиотеки OpenCV и OpenVINO.

Вы можете указать свою нейронную сеть, для этого при запуске укажите её имя в следующем виде:
    TestOpVnn.exe --model your_network
, в этом случае будет использоваться нейронная сеть из файлов:
    your_network.bin
	your_network.xml

Приложение применяет нейронную сеть ко всем файлам из папки input_folder и помещает результаты
в папку output_folder. Также в конце работы приложение напечатает небольшой отчёт, в котором
укажет:
    количество распознанных изображений
	общее и среднее время, затраченное на распознавание
	общее и среднее время, затраченное на помещение изображений в блоб нейронной сети
	общее и среднее время, затраченное на обработку (включая загрузку и сохранение изображения)

В папке /data/ сохранены примеры входа и ожидаемого выхода для нейронной сети по умолчанию.

Также вы можете запустить нейронную сеть на нескольких ядрах. Для этого укажите к командной
строке соответствующий параметр:
    TestOpVnn.exe --cores cores_number
, например, чтобы запустить распознавание на 2-х ядрах, используйте:
    TestOpVnn.exe --cores 2

Вы можете комбинировать параметры --cores и --model:
    TestOpVnn.exe --cores cores_number
    TestOpVnn.exe --model your_network
    TestOpVnn.exe --cores cores_number --model your_network
    TestOpVnn.exe --model your_network --cores cores_number

Не забывайте о том, что данное приложение является демонстрационным, поэтому качество
распознавания может быть очень плохим (сеть обучалась не очень долго). Для промышленного
применения данное приложение непригодно.
